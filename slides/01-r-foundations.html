R Foundations

Introduction

Meaningful data analysis requires the use of computer software.

R statistical software is popular for data analysis in:

  • academia
  • industry
  • government

We will lay a basic foundation of R knowledge and skills in this module.

What is R?

R is:

  • A programming language and environment designed for statistical computing.
  • Free, open source, and runs on many different types of computers.
  • An interactive programming language.
  • Highly extendable through packages and writing your own functions.

Installing R

Download an installer program from the R Project’s website (https://www.r-project.org/) from https://cloud.r-project.org/.

  • A stable link for the most current installer program is available at https://cloud.r-project.org/bin/windows/base/release.html.
  • Similar links are not currently available for Mac and Linux computers.

Installing R Studio

R Studio Desktop is a free “front end” for R provided by R Studio (https://rstudio.com/).

R Studio Desktop makes doing data analysis with R much easier by adding an Integrated Development Environment (IDE) and providing many other features.

Download R Studio at https://rstudio.com/products/rstudio/download/.

R Studio Layout

R Studio Desktop has four panes:

  1. Console: the pane where commands are run.
  2. Source: the pane where you prepare commands to be run.
  3. Environment/History: the pane where you can see all the objects in your workspace, your command history, and other information.
  4. The Files/Plot/Packages/Help: the pane where you navigate between directories, where plots can be viewed, where you can see the packages available to be loaded, and where you can get help.
